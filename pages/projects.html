Projects

Welcome! This page is a glimpse into my journey in computer science. From learning the basics of web development to exploring programming languages, I'm excited to share my progress with you. Dive in to discover more about my projects and interests.

1. Responsive web design

This project exemplifies my ability to create responsive web pages using HTML and CSS. Through media queries, the layout adjusts effortlessly to various screen sizes, guaranteeing an optimal viewing experience on desktops, tablets, and smartphones.

2. The Mondrian Composition

In this project, I recreated a Mondrian-style composition using CSS grid. By leveraging CSS grid's layout capabilities, I arranged colorful blocks to mimic the abstract geometric patterns characteristic of Mondrian's artwork.

3. Moveit

This project is a simple website for a moving company built using Bootstrap. Leveraging Bootstrap's pre-designed components and responsive grid system, I created a user-friendly interface that highlights the company's services, contact information, and testimonials.

4. The dice game

This project is a simple website featuring a dice game implemented using JavaScript. Each time the webpage is refreshed, two dice are rolled, and the winner is determined based on the higher number rolled.

5. The Drum Kit

This project is a website featuring a virtual keyboard drumstick, where pressing buttons triggers different drumstick sounds corresponding to different alphabets. Through JavaScript, each button press activates a unique sound, creating an interactive and entertaining experience reminiscent of playing drums with keyboard keys

6. The Simon game

This project is a digital version of the classic Simon game, featuring four buttons that light up in a sequence. Players must repeat the sequence by clicking the buttons in the same order. With JavaScript, the game tracks and generates increasingly complex patterns, challenging players' memory and reflexes.
